# Sigma Octantis Website

Sigma Octantis is an independent digital studio building small digital experiences for self-trust, agency, and better choices in an AI-saturated world.

The studio’s first product direction is **First Answer**: a private ritual for writing your own answer before asking AI, friends, social media, or the internet.

Core thesis:

> Hear yourself first.

This repository contains the public website for Sigma Octantis.
